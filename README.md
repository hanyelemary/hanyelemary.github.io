## Getting started

### Prerequisites
* [NodeJS](https://nodejs.org/en/)
* [nvm (Node Version Manager)](https://github.com/nvm-sh/nvm)

### Set Nodejs version
```
nvm use
```

### Install dependencies
```
npm i
```

### Build
```
npm run build
```

### Run
Open up `index.html` in the browser.
